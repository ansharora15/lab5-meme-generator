Magic 8-Ball and Fortune Cookie Project
========================================

This project combines the Magic 8-Ball with Fortune Cookie sayings.
Users can ask a question and receive a random response from either a
Magic 8-Ball answer list or a Fortune Cookie sayings list.

Features:
- Provides responses from both Magic 8-Ball and Fortune Cookie sayings.
- Easy-to-use interface with a text input for the user's question.
- Randomly selects an answer from combined lists of sayings.

How to Use:
1. Open the "index.html" file in your web browser.
2. Type your question in the input field.
3. Click the "Ask" button.
4. The answer will be displayed on the screen.

Files Included:
- index.html : Main HTML file.
- css/style.css : Styles for the web page.
- js/script.js : JavaScript logic for the responses.
